## 配色说明

### V6.2 更新说明
1. **科技蓝 #0066CC** - 经典配色，体现科技感与专业性
2. **深空灰 #5A5A61** - 商务风格，适合办公场景
3. **珊瑚红 #FF7359** - 活力风格，吸引年轻用户

### 渲染设置参考
- 科技蓝材质：Base Color (0.0, 0.4, 0.8, 1.0), Roughness 0.3, Metallic 0.1
- 深空灰材质：Base Color (0.35, 0.35, 0.38, 1.0), Roughness 0.3, Metallic 0.1
- 珊瑚红材质：Base Color (1.0, 0.45, 0.35, 1.0), Roughness 0.3, Metallic 0.1

### 生产建议
- 建议外壳表面采用哑光处理，减少反光
- 三种配色已渲染验证，可用于后续生产
